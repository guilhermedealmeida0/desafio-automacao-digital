# Sistema de Automação de Qualidade

## Descrição

Este projeto simula um sistema de controle de qualidade em uma linha de produção industrial.

O sistema recebe dados de peças e realiza a validação automática com base em critérios definidos.

## Funcionalidades

- Cadastro de peças
- Validação automática (peso, cor e comprimento)
- Classificação em aprovadas e reprovadas
- Organização em caixas
- Relatório final

## Como executar

1. Abra o terminal
2. Execute o comando:

```bash
python desafio_automacao.py
```
